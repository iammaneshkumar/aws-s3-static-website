# AWS S3 Static Website Project

This is a simple static website hosted on AWS S3 Free Tier.

## Project Description
- This website demonstrates basic HTML and CSS skills.
- Fully responsive and works on desktop and mobile devices.
- Hosted publicly using AWS S3 static website hosting.

## Files
- `index.html` - main webpage
- `styles.css` - styling for the website
- `screenshot/` - Website view

## AWS S3 Setup
1. Static website hosting enabled in S3.
2. Public access allowed via bucket policy.
3. Files uploaded to the S3 bucket.
